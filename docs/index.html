Bienvenido a Flicks Time

Flicks Time es una aplicación desarrollada en Flutter que te permite explorar y descubrir películas y series. Utiliza la API de The Movie Database (TMDB) para proporcionar información detallada sobre tus contenidos favoritos.

Características Principales

  • Explora una amplia base de datos de películas y series.
  • Consulta detalles como sinopsis, elenco, calificaciones y más.
  • Disfruta de una experiencia visual mejorada con elementos multimedia de Storyset y Freepik.
